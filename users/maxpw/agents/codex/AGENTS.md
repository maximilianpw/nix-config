# Codex Routing

Use Claude for:

- UI/UX judgment
- Copywriting
- API design taste checks
- Product/design-heavy plan review
- High-signal implementation review

Invoke with:

- `claude -p --model fable` for strongest taste/intelligence
- `claude -p --model opus` for strong review
- `claude -p --model sonnet` for cheaper routine input

Ask Claude for critique or structured recommendations by default; ask for edits only in isolated worktrees or tightly scoped patches. Use `--output-format json` or `--json-schema` when structured output matters.
